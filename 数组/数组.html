数组是最基础的数据结构之一，存储在内存中的连续块中。数组中的元素可以通过索引直接访问。

示例题目 1：两数之和 (Two Sum)

=====
  • 题目： 给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。
    示例:
    给定 nums = [2, 7, 11, 15], target = 9
因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]
  • 解答：
    def two_sum(nums, target):
    hashmap = {}
    for index, num in enumerate(nums):
        complement = target - num
        if complement in hashmap:
            return [hashmap[complement], index]
        hashmap[num] = index
    这段代码利用哈希表存储已经访问过的数的值和索引，然后在遍历数组时查找是否存在符合条件的数对。

示例题目 2：移动零 (Move Zeroes)

=====
  • 题目： 给定一个数组 nums，编写一个函数将所有的0移动到数组的末尾，同时保持非零元素的相对顺序。
    示例:
    输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
  • 解答：
    def move_zeroes(nums):
    last_non_zero_found_at = 0
    for current in range(len(nums)):
        if nums[current] != 0:
            nums[last_non_zero_found_at], nums[current] = nums[current], nums[last_non_zero_found_at]
            last_non_zero_found_at += 1
    该解法通过双指针法实现，维护一个指针指向最后一个非零元素的位置，不断交换零元素和非零元素的位置。
