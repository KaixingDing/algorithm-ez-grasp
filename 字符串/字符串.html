字符串是编程中的基本数据类型之一，在算法面试中也经常会涉及到各种字符串操作和处理问题。字符串问题通常包括查找、替换、解析、匹配等操作。

字符串的基本操作

在Python中，字符串是不可变的序列类型。以下是一些常见的字符串操作：
  1. 创建字符串：
    s = "Hello, World!"
  2. 访问字符串：
    print(s[0])     # 输出: H
print(s[-1])    # 输出: !
  3. 字符串切片：
    print(s[0:5])   # 输出: Hello
print(s[:5])    # 输出: Hello
print(s[7:])    # 输出: World!
  4. 字符串连接：
    s1 = "Hello"
s2 = "World"
s = s1 + ", " + s2 + "!"  # 输出: Hello, World!
  5. 字符串查找：
    print(s.find("World"))    # 输出: 7
print(s.index("World"))   # 输出: 7
  6. 字符串替换：
    s = s.replace("World", "There")  # 输出: Hello, There!
  7. 字符串分割：
    parts = s.split(", ")    # 输出: ['Hello', 'There!']
  8. 去除空白：
    s = "  Hello, World!  "
s = s.strip()            # 输出: Hello, World!

示例题目 1：最长无重复子串 (Longest Substring Without Repeating Characters)

题目描述

  • 题目： 给定一个字符串 s，请你找出其中不含有重复字符的最长子串的长度。
    示例：
    输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

解答

我们可以使用滑动窗口和哈希表来高效解决这个问题。通过移动窗口的左右边界来保持窗口内的字符没有重复。
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        char_index = {}
        left = 0
        max_length = 0
        
        for right, char in enumerate(s):
            if char in char_index and char_index[char] >= left:
                left = char_index[char] + 1
            char_index[char] = right
            max_length = max(max_length, right - left + 1)
        
        return max_length

# 测试样例
solution = Solution()
print(solution.lengthOfLongestSubstring("abcabcbb"))  # 输出: 3
print(solution.lengthOfLongestSubstring("bbbbb"))     # 输出: 1
print(solution.lengthOfLongestSubstring("pwwkew"))    # 输出: 3

示例题目 2：字符串转换整数 (String to Integer, Atoi)

题目描述

  • 题目： 请你来实现一个 myAtoi 函数，使其能将字符串转换成一个32位有符号整数。
    示例：
    输入: s = "42"
输出: 42

输入: s = "   -42"
输出: -42
解释: 第一个非空白字符为 '-'，它是一个负号。我们尽可能将负号与后面所有连续的数字组合起来，最后得到 -42 。

解答

我们可以按步骤处理字符串，跳过前导空格，处理符号，然后逐位解析数字，并处理整数范围溢出。
class Solution:
    def myAtoi(self, s: str) -> int:
        s = s.strip()
        if not s:
            return 0
        
        sign = 1
        if s[0] in ['-', '+']:
            if s[0] == '-':
                sign = -1
            s = s[1:]
        
        result = 0
        for char in s:
            if not char.isdigit():
                break
            result = result * 10 + int(char)
            # 检查是否超过32位整数范围
            if result > 2147483647:
                return 2147483648 if sign == -1 else 2147483647
        
        return sign * result

# 测试样例
solution = Solution()
print(solution.myAtoi("42"))        # 输出: 42
print(solution.myAtoi("   -42"))    # 输出: -42
print(solution.myAtoi("4193 with words"))  # 输出: 4193
print(solution.myAtoi("words and 987"))    # 输出: 0
print(solution.myAtoi("-91283472332"))     # 输出: -2147483648

示例题目 3：验证回文字符串 (Valid Palindrome)

题目描述

  • 题目： 给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。
    示例：
    输入: s = "A man, a plan, a canal: Panama"
输出: true

输入: s = "race a car"
输出: false

解答

我们可以使用双指针来检查字符串是否为回文串，忽略非字母数字字符和大小写。
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left, right = 0, len(s) - 1
        while left < right:
            while left < right and not s[left].isalnum():
                left += 1
            while left < right and not s[right].isalnum():
                right -= 1
            if s[left].lower() != s[right].lower():
                return False
            left += 1
            right -= 1
        return True

# 测试样例
solution = Solution()
print(solution.isPalindrome("A man, a plan, a canal: Panama"))  # 输出: True
print(solution.isPalindrome("race a car"))                      # 输出: False
