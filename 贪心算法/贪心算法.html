贪心算法（Greedy Algorithm）是一种在每一步选择中都采取当前最优解的策略，以期望最终得到全局最优解的算法。贪心算法通常用于解决优化问题，例如最小生成树、最短路径、任务调度等。虽然贪心算法并不总是能保证得到全局最优解，但在某些特定问题中，它能提供高效且正确的解决方案。

贪心算法的基本概念

基本步骤

  1. 选择当前最优解：在当前状态下，选择局部最优解。
  2. 检查是否完成：判断是否已经达到问题的结束条件。
  3. 更新状态：更新当前状态，继续选择下一步最优解。

示例题目 1：分发饼干 (Assign Cookies)

题目描述

  • 题目： 假设你是一位很棒的家长，你想要给你的孩子们一些饼干。但是，每个孩子只能给一块饼干。对每个孩子 i，都有一个胃口值 g[i]，每块饼干 j，都有一个尺寸 s[j]。如果 s[j] >= g[i]，我们可以将这个饼干分配给孩子 i，这个孩子会得到满足。你的目标是尽可能满足越多数量的孩子，并输出这个最大数值。
    示例：
    输入: g = [1,2,3], s = [1,1]
输出: 1
解释: 你有三个孩子和两块小饼干，其中只有一种方案可以让一个孩子得到满足。

解答

我们可以使用贪心算法来解决这个问题。通过将饼干和孩子的胃口数组排序，然后依次为每个孩子分配最小的能满足其胃口的饼干。
from typing import List

class Solution:
    def findContentChildren(self, g: List[int], s: List[int]) -> int:
        g.sort()
        s.sort()
        child_i = 0
        cookie_j = 0
        while child_i < len(g) and cookie_j < len(s):
            if s[cookie_j] >= g[child_i]:
                child_i += 1
            cookie_j += 1
        return child_i

# 测试样例
solution = Solution()
print(solution.findContentChildren([1,2,3], [1,1]))  # 输出: 1
print(solution.findContentChildren([1,2], [1,2,3]))  # 输出: 2

示例题目 2：跳跃游戏 (Jump Game)

题目描述

  • 题目： 给定一个非负整数数组 nums，你最初位于数组的第一个位置。数组中的每个元素代表你在该位置可以跳跃的最大长度。判断你是否能够到达最后一个位置。
    示例：
    输入: nums = [2,3,1,1,4]
输出: True
解释: 我们可以先跳 1 步，从位置 0 到达位置 1，然后再从位置 1 跳 3 步到达最后一个位置。

输入: nums = [3,2,1,0,4]
输出: False
解释: 无论怎样，都无法跳到最后一个位置。

解答

我们可以使用贪心算法来解决这个问题。通过维护一个最大可达位置 max_reachable，在遍历数组时不断更新这个位置，并检查是否可以到达最后一个位置。
from typing import List

class Solution:
    def canJump(self, nums: List[int]) -> bool:
        max_reachable = 0
        for i, length in enumerate(nums):
            if i > max_reachable:
                return False
            max_reachable = max(max_reachable, i + length)
        return True

# 测试样例
solution = Solution()
print(solution.canJump([2, 3, 1, 1, 4]))  # 输出: True
print(solution.canJump([3, 2, 1, 0, 4]))  # 输出: False

示例题目 3：分隔字符串使同种字符出现在一起 (Partition Labels)

题目描述

  • 题目： 字符串 S 由小写字母组成。我们要把这个字符串分成尽可能多的片段，同一个字母只会出现在其中的一个片段。返回一个表示每个片段大小的列表。
    示例：
    输入: S = "ababcbacadefegdehijhklij"
输出: [9,7,8]
解释:
划分结果为 "ababcbaca", "defegde", "hijhklij"。
每个部分的长度分别是9, 7, 8。

解答

我们可以使用贪心算法来解决这个问题。通过找到每个字符的最远出现位置，然后在遍历字符串时划分片段。
from typing import List

class Solution:
    def partitionLabels(self, S: str) -> List[int]:
        last_index = {char: i for i, char in enumerate(S)}
        j = anchor = 0
        result = []
        
        for i, char in enumerate(S):
            j = max(j, last_index[char])
            if i == j:
                result.append(i - anchor + 1)
                anchor = i + 1
        
        return result

# 测试样例
solution = Solution()
print(solution.partitionLabels("ababcbacadefegdehijhklij"))
# 输出: [9, 7, 8]
